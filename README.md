# 櫻坂46 四期生LIVE リンク集

LaLa arena TOKYO-BAYで開催される櫻坂46「四期生LIVE」来場者向けの公式リンク集です。

天気・台風・交通情報の独自取得は行わず、会場、櫻坂46公式、最寄り路線、駅、バス、空港、地図への導線だけをまとめます。

## 構成

- `index.html`: リンク集本体
- `styles.css`: モバイル優先のカードUI
- `app.js`: カテゴリフィルタ

## 掲載カテゴリ

- 鉄道: JR京葉線、JR武蔵野線、京成本線、南船橋駅、船橋競馬場駅
- 公式: 櫻坂46公式、四期生LIVE特設サイト、LaLa arena公式
- バス: ららぽーとTOKYO-BAYバス案内、京成電鉄バスグループ路線バス
- 空港: 成田空港、羽田空港、京成スカイライナー、京急
- 地図: 会場Google Map、南船橋駅からの徒歩ルート、船橋競馬場駅からの徒歩ルート

## ローカル確認

```bash
php -S localhost:4173
```

ブラウザで `http://localhost:4173/` を開きます。

## 方針

非公式サイトとして、来場者が公式ページへ最短で移動できることを優先します。開催可否、入場ルール、運行状況は各公式ページの情報を確認してください。
